Бд - mysql
Есть сидеры и фабрики, php artisan db:seed для заполнения бд
В моделях использован soft delete
для обновления/создания учебного плана формат json:
{
   "lectures":[
       {
           "id": 2,
           "order": 1
       },
       {
           "id": 2,
           "order": 2
       }
   ]
}
порядок нужен для разного порядка лекций
UniqueLectureRule - для валидации, проверяем, чтобы для учебного плана все лекции были уникальны
